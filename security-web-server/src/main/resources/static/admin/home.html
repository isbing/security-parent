说明：

spring security 全注解式的权限管理

动态配置菜单、权限、角色以及用户

可以在页面中 精确控制 某个按钮的出现

内置功能：

菜单管理：添加一级 二级菜单、查询菜单等

权限管理：添加权限、权限属于哪一个菜单

角色管理：角色查询、添加角色、给角色分配权限

用户管理：用户查询、添加用户、修改用户、给用户分配角色

其他说明：

使用lombok简化java代码，让源码更简洁，可读性高

当前版本还是根据session、下面两条为未实现的功能

采用token进行权限校验，禁用session，未登录返回401，权限不足返回403

采用redis存储token及权限信息
